Кейс 8

Мобильное приложение для аптечных покупок на каждый день

Описание проекта

Мобильное приложение для удобного заказа лекарств, косметики и товаров для здоровья с доставкой на дом.

Задача: создать простой и понятный интерфейс для ежедневного использования, где покупка привычных товаров занимает минимум времени.

Решение: разработка дизайн-макета интуитивного каталога с крупными категориями, системы повтора прошлых заказов в один клик и наглядного статуса доставки. Дизайн выполнен в светлых тонах, чтобы процесс выбора был максимально комфортным.

Инструменты: Figma, UX Research, UI Design System, Mobile Adaptive Layout, Prototyping.
